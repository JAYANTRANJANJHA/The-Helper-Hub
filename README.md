## ğŸ Getting Started

### Prerequisites

- Node.js (v18.x recommended)
- npm or yarn

### Quick Setup

Clone and enter the repository:

```bash
git clone https://github.com/JAYANTRANJANJHA/The-Helper-Hub.git
cd The-Helper-Hub
```

### Install & Run (Frontend)

```bash
cd client
npm install
npm run dev
```

### Install & Run (Backend)

```bash
cd ../server
npm install
npm start
```
